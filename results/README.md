# Results

The population artifact from Drive contained 24 PNG figures and one CSV with 72
protein-level rows. They are unpacked here.

- `tables/results_population.csv`: original result table.
- `tables/population_summary.csv`: derived median table.
- `metrics/evaluation_summary.json`: machine-readable derived summary.
- `figures/`: original population and case-study PNG files.

No values in the derived summary were manually copied from the paper.
